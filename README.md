# Simple Indexer

This is a simple indexer for initia.

# How to use

```bash
npm i
```

```bash
npm run start
```

# Configuration

| Name                    | Description                                    | Default                               |
|-------------------------|------------------------------------------------|---------------------------------------|
| SERVER_PORT             | Server port                                    | 6000                                  |
| L1_LCD_URL              | LCD URL                                        | 'http://localhost:1317'               |
| L1_RPC_URL              | RPC URL                                        | 'http://localhost:26657'              |
