# Simple Indexer

This is a simple indexer for initia.

# How to use

```bash
npm i
```

```bash
npm run start
```

# Configuration

| Name                    | Description                                    | Default                               |
|-------------------------|------------------------------------------------|---------------------------------------|
| SERVER_PORT             | Server port                                    | 6000                                  |
| LCD_URL                 | LCD URL                                        | 'http://localhost:1317'               |
| RPC_URL                 | RPC URL                                        | 'http://localhost:26657'              |
